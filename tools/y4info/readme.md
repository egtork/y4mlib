# y4info

y4info gathers information about a y4m file and prints it in a human readable format.

### Example

```
> ./y4info aspen.y4m

Stream header information:
  Width: 1920
  Height: 1080
  Frame rate: 30000:1001
  Interlacing: p
  SampleAspectRatio: 1:1
  Chroma: 422
  Metadata: []
Frames:
  570
Duration:
  19.019s
```
